# Installation


1. Set up git with your user name and email
2. Open a terminal/shell and type:
    ```
    * git config --global user.name (like "smith.name")
    * git config --global user.email (like "smith@email.com")
    * git init
    * git clone /path/to/repository (Go to git account copy the http link of clone,this is create a working copy of a local repository)
    * git status ( It's check which branch you present)
      ( Note: If you create a new branch for your code then write this syntax: git checkout -b <branchname> 
        After create your branch go to your own branch in terminal then follow the following step)
    * git add <filename>
            or
      git add *
            or
      git add .
    * git commit -m "Commit message" (There you write your short message which things you add in your git)
    * git push origin master ( There master is branch name, which branch you present write there those branch name)
    ```
